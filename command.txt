git init	Создать новый репозиторий: git init myproject
git clone	Клонировать репозиторий: git clone https://github.com/user/repo.git
git status	Показать статус изменений
git log	История коммитов: git log --oneline --graph
git diff	Показать изменения: git diff, git diff --staged
git add	Добавить файлы в индекс: git add file.txt, git add .
git restore	Откатить изменения: git restore file.txt, git restore --staged file.txt
git rm	Удалить файл: git rm file.txt, git rm --cached file.txt
git commit	Создать коммит: git commit -m "message"
git commit --amend	Исправить последний коммит: git commit --amend -m "new message"
git branch	Работа с ветками: git branch, git branch feature, git branch -d feature
git checkout	Переключиться: git checkout main, git checkout -b new-branch
git switch	Переключиться (новый): git switch main, git switch -c new-branch
git merge	Слить ветки: git checkout main && git merge feature
git reset	Отменить коммиты: git reset --soft HEAD~1, git reset --hard HEAD~1
git revert	Отменить коммит: git revert HEAD, git revert abc1234
git remote	Управление удалёнными: git remote -v, git remote add origin url
git fetch	Загрузить изменения: git fetch origin
git pull	Загрузить и слить: git pull origin main
git push	Отправить изменения: git push origin main, git push -u origin main
git stash	Временно сохранить: git stash, git stash pop, git stash list
git show	Показать коммит: git show abc123, git show HEAD
git reflog	История всех действий
git grep	Поиск в коде: git grep "function"
git config	Настройка: git config --global user.name "Name", git config --list
Добавить в ~/.gitconfig	[alias] s = status, co = checkout, br = branch, ci = commit
git tag	Работа с тегами: git tag v1.0, git tag -a v1.0 -m "Version 1.0"
git cherry-pick	Взять коммит: git cherry-pick abc123
git clean	Удалить неотслеживаемые: git clean -fd, git clean -n (показать что удалит)
Ключевые опции
Для git log: 
--oneline — кратко, --graph — с графиком
-p — с изменениями,--since="2 days ago" — за период
-n 5 — последние 5 коммитов
Для git diff:
•	git diff — изменения не в staged
•	git diff --staged — изменения в staged
•	git diff HEAD~3..HEAD — сравнить диапазон
Для git reset:
•	--soft — отменить коммит, изменения в staged
•	--mixed — отменить коммит, изменения в рабочей директории
•	--hard — отменить коммит и удалить изменения
1.	Обновить и слить: git pull --rebase origin main
2.	История с ветками: git log --oneline --graph --all
3.	Что изменилось: git diff --name-status origin/main
4.	Исправить последний коммит: git commit --amend --no-edit (без изменения сообщения)
5.	Переместить ветку: git branch -f main HEAD~3 (переместить main на 3 коммита назад)
